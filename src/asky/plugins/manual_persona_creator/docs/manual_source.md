# Manual Sources

Manual sources allow you to expand a persona's knowledge with third-party or short-form content.

## Supported Kinds
- **Biography / Autobiography**: Extensive life details and timeline events.
- **Interview**: Persona-attributed knowledge with speaker-role metadata.
- **Article / Essay / Speech / Posts / Notes**: Authored viewpoints and facts.

## Review and Promotion
- **Pending**: Biographies and interviews require manual review and approval before they influence persona answers.
- **Auto-Project**: Authored primary short-form content (articles, posts) projects immediately into the catalog.

## Path Semantics
You can point to a single file or a directory. The system will identify supported files and prepare an ingestion job.

## When to choose
Use manual sources for any content that isn't a single high-fidelity authored book but still provides valuable evidence for the persona's knowledge base.
